Type of form Elements

HTML offers various types of form elements to collect different types of user input. Here are some commonly used form elements:

1. Text Input: Allows users to input a single line of text.

2. Password Input: Conceals the characters typed by the user, often used for sensitive information.

3. Textarea: Allows users to input multiple lines of text.

4. Checkbox: Allows users to select one or more options from a list.

5. Radio Buttons: Allows users to select one option from a list.
